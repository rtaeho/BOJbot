# 🎯 BOJ 인증 봇

> 백준 알고리즘 문제 풀이를 친구들과 함께 인증하고 경쟁하는 카카오톡 챗봇

[![GitHub](https://img.shields.io/badge/GitHub-rtaeho/BOJbot-blue)](https://github.com/rtaeho/BOJbot)

---

## 📌 주요 기능

### ✅ 백준 아이디 등록
간편하게 백준 계정을 연동하여 자동 인증 시작

### 📊 실시간 그룹 랭킹
그룹원 중 누가 가장 많이 풀었는지 실시간 순위 확인

### 🔥 연속 풀이 스트릭
매일 문제 풀이 기록을 추적하여 꾸준함을 독려

### 📈 일일 인증 현황
오늘 그룹원 전체의 풀이 현황을 한눈에 확인

### ⏰ 자동 현황 알림
매일 밤 11시 자동으로 오늘의 인증 결과 발표

---

## 🎮 사용 방법

### 카카오톡 채널에서

```
등록 백준아이디
```
백준 아이디를 등록하세요

```
현황
```
오늘의 인증 현황을 확인하세요

## 💡 활용 시나리오

- 🎓 **대학 알고리즘 스터디** - 팀원들과 매일 문제 풀이 인증
- 💼 **코딩테스트 준비반** - 취업 준비생들의 꾸준한 학습
- 👥 **개발자 친구 모임** - 매일 1문제씩 함께 도전
